# API
> Rust GraphQL API [youpidok/cs-rust-flutter](https://hub.docker.com/repository/docker/youpidok/cs-rust-flutter/general)

## Install
**Config en vars**
```shell
cp .env.dist .env
```

**Watch**
```shell
cargo watch -x run
```

**Build**
```shell
cargo build
```
